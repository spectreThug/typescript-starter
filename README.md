# Installation

Either through cloning with git or by using [npm](http://npmjs.org) (the recommended way):

```bash
$ npm install -g @thug74/typescript-starter # or using yarn: yarn global add @thug74/typescript-starter
```

# Usage

```bash
$ typescript-starter
```
